# To-do App

A full-stack Todo App built with **React** (frontend) and **Node.js + Express + MongoDB**.  
Features: User signup/login (JWT), protected routes, add/update/delete todos, task tracking.

---

## Setup Instructions

### Prerequisites
- Node.js & npm
- MongoDB (Atlas or local)

---

### Backend
1. Go to backend folder:  
```bash
cd backend
install dependencies
npm install
Install dependencies:

npm install


Create .env file:

PORT=5000
MONGODB_URL=<your MongoDB connection string>
JWT_SECRET=<your JWT secret>


Start backend server:

npm start


Runs on http://localhost:5000

Frontend

Go to frontend folder:

cd frontend


Install dependencies:

npm install


Create .env file:

VITE_API_URL=http://localhost:5000


Start frontend server:

npm run dev


Runs on http://localhost:5173

Usage

Sign up and login

Add, edit, delete todos

Track task status
